MULTIPLE LINEAR REGRESSION - MULTI-CHANNEL MARKETING ANALYSIS

PROJECT OVERVIEW
This project uses Multiple Linear Regression to examine the impact of different marketing channels on Sales. The analysis includes data cleaning, exploratory data analysis, multicollinearity testing, model development, diagnostic checks, and business recommendations.

DATASET
The dataset contains the following variables:
1.     TV
2.     Radio
3.     Social Media
4.     Influencer
5.     Sales

Categorical variables were encoded before model development.

OBJECTIVES
1. Explore and clean the dataset.
2. Detect multicollinearity using VIF.
3. Build a Multiple Linear Regression model using Statsmodels.
4. Evaluate model performance using R - squared and Adjusted R- squared
5. Validate regression assuptions using diagnostic plots.
6. Interpret coefficients and p - values.
7. Provide business recommendations based on the findings.

   TOOLS USED
   1. Python
   2. Pandas
   3. Numpy
   4. Matplotlib
   5. Seaborn
   6. Statsmodels
   7. Scipy

KEY FINDINGS
- The regression model achieved an R- squared of approximately 0.904
- Radio advertising had a statistically significant positive effect on sales.
- Low TV advertising was associated with lower Sales.
- Social Media and Influencer variables were not statistically significant in this model.

 BUSINESS RECOMMENDATION
 Increase investment in Radio advertising, maintain effective TV advertising levels, and review the effectiveness of social Media and influencer campaigns before increasing their budgets.
